📖 Erklärung: Das Perfekt

Das Perfekt ist eine Vergangenheitsform im Deutschen. Sie wird mit „haben" oder „sein" + Partizip II gebildet.

Perfekt-Struktur

Formel: haben/sein + Partizip II

  • Position 2: haben/sein (konjugiert)
  • Satzende: Partizip II

Beispiel:

Ich habe ein Buch gelesen.
I have read a book.

Partizip II: Regelmäßig

Regel: ge- + Stamm + -(e)t

  • machen → gemacht
  • spielen → gespielt
  • kaufen → gekauft
  • kochen → gekocht
  • lernen → gelernt

Partizip II: Unregelmäßig

Regel: ge- + Stamm + -en (oft mit Lautwechsel)

  • schreiben → geschrieben
  • lesen → gelesen
  • sehen → gesehen
  • fahren → gefahren
  • essen → gegessen

Verben mit „sein"

Bewegungsverben und Zustandsveränderungen nutzen „sein":

  • gehen → gegangen
  • kommen → gekommen
  • fahren → gefahren
  • fliegen → geflogen
  • bleiben → geblieben
Louise ist nach Berlin gefahren.
Louise drove to Berlin.

Trennbare Verben

Präfix + ge- + Stamm + -(e)t/-en

Das „ge-" kommt zwischen Präfix und Stamm.

  • aufstehen → aufgestanden
  • einkaufen → eingekauft
  • anrufen → angerufen

Verben auf -ieren

Kein „ge-"! Regel: Stamm + -t

  • telefonieren → telefoniert
  • studieren → studiert
  • fotografieren → fotografiert
Ich habe mit Mama telefoniert.
I called mom.
